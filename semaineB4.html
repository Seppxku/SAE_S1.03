1. Installation d'un service web.

Pour pouvoir installer notre poste de développement, il faut créer au préalable un serveur qui hébergera notre site.

Mais, lequel ?

Nous avons fait le choix d’installer un serveur web Apache, un logiciel libre qui permet de diffuser des pages web sur Internet. Pourquoi ? :

  • Apache a une grande communauté et une documentation très détaillée et large.
  • Apache est un logiciel libre.
  • Son support est étendu sur différents systèmes d'exploitation.

  • -c active la coloration de la sortie pour rendre les informations plus lisibles dans le terminal
  • addr affiche des détails sur les interfaces réseau disponibles, les adresses IP assignées et les états des interfaces.
